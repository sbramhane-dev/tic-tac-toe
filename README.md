# Tic Tac Toe Game 🎮

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Two-player mode (X vs O)
- Winner detection
- Draw detection
- Reset and New Game functionality

##Languages Used
- HTML
- CSS
- JavaScript

## Live Demo
https://sbramhane-dev.github.io/tic-tac-toe/

## How to Run
Open index.html in your browser
